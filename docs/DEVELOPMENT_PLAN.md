## 部署
项目已配置为 PWA，可部署到 Vercel：
1. 将项目推送到 GitHub
2. 在 Vercel 中导入仓库
3. 设置环境变量 VITE_SUPABASE_URL 和 VITE_SUPABASE_ANON_KEY
4. 部署
