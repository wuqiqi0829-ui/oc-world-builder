# 功能需求规格

## 模块 A：基础设施
- **A1 用户系统**：邮箱注册、登录、找回密码。Supabase Auth 实现。
- **A2 云端同步**：所有数据实时同步到 Supabase，多设备自动一致。
- **A3 多世界观管理**：新建/重命名/删除/切换。数据按 world_id 隔离。
- **A4 世界观卡片**：名称 + 封面图 + 简介。卡片网格或列表两种视图。
- **A5 暗黑模式**：亮/暗主题一键切换，CSS 变量驱动。
- **A6 数据导出/导入**：全量 JSON 导出下载，JSON 文件导入恢复。
- **A7 单世界观导出**：可选 JSON、Markdown 文本或长图导出。
- **A8 打印支持**：@media print 样式，隐藏交互元素。
- **A9 响应式适配**：PC/平板/手机三断点自适应。

## 模块 B：通用编辑能力
- **B1 富文本**：TipTap 实现，支持加粗/斜体/列表/引用/分割线/复制粘贴。
- **B2 拖拽排序**：dnd-kit 实现，卡片/列表项/段落可拖拽。
- **B3 自动保存**：编辑内容 debounce 1.5s 自动保存到 Supabase。
- **B4 图片管理**：上传/粘贴/拖拽添加图片，Supabase Storage 存储。
- **B5 图片预览**：点击放大灯箱，支持备注。
- **B6 图片压缩**：browser-image-compression 前端压缩。

## 模块 C：时间线
- **C1** 节点 CRUD（标题、时间标签、富文本描述、配图）
- **C2** 拖拽排序，正序/倒序切换
- **C3** 节点折叠/展开
- **C4** 可视化纵向时间轴

## 模块 D：地图
- **D1** 底图上传，缩放平移
- **D2** 可拖拽标记点，点击查看详情
- **D3** 地点详情（名称/简介/配图/关联人物/区域）
- **D4** 分类筛选，列表地图联动

## 模块 E：人物设定库
- **E1** OC 人设卡 CRUD
- **E2** 字段：姓名/昵称/性别/年龄/外貌/性格/背景/能力/弱点/口头禅
- **E3** 多图上传（立绘/头像/细节），拖拽排序
- **E4** 关联势力/职业/时间节点
- **E5** 卡片列表 + 筛选

## 模块 F：职业/势力/种族设定
- **F1** 自定义分类，条目管理
- **F2** 富文本+配图，关联人物
- **F3** 自定义字段模板

## 模块 G：组织/势力
- **G1** 国家/帮派/公会等组织管理
- **G2** 成员归属

## 模块 H：物品图鉴
- **H1** 物品 CRUD（图片/属性/背景故事）
- **H2** 分类管理

## 模块 I：关系图谱
- **I1** 人物间关系定义（亲友/敌对/师徒等）
- **I2** React Flow 可视化，拖拽布局
- **I3** 跨模块总览

## 模块 J：标签系统
- **J1** 全局自定义标签（含颜色）
- **J2** 给世界观/人物/事件打标签
- **J3** 标签筛选

## 模块 K：主线剧情
- **K1** 多条故事线管理
- **K2** 章节列表（新增/排序/删除）
- **K3** 关联角色/地点/事件

## 模块 L：灵感速记
- **L1** 快速笔记 CRUD
- **L2** 后续关联到具体模块

## 模块 M：全局搜索
- **M1** 跨模块关键词搜索
- **M2** 跨世界观搜索
